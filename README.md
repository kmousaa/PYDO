# What Is It?

Welcome to **PYDO**, your typical to-do list… except it’s in the terminal.  
And it has a slightly too helpful AI assistant called **TOODLES**.

# What Can You Do?

- Create various checklists for different parts of life  
  *(work, shopping, shows to binge, whatever)*

- Switch between checklists and view them all  

- Add your TODOs, tick them off, or delete them  

<img width="681" height="446" alt="image" src="https://github.com/user-attachments/assets/c38e70d8-f2fe-4504-8498-80562f25f47c" />

- Ask **TOODLES** for help  
  - Add tasks  
  - Remove tasks  
  - Mark tasks complete / uncomplete  
  - Rearrange your list  
  - Ask questions about your data (`do tell ...`)  

- Uses `llama-3.3-70b-versatile` for AI features  

- Checklists are saved per session  

- Type `help` on any screen to see commands

# Alright, let’s DO it?

Here is what you need to do:

- [ ] Clone the repo
```bash
git clone https://github.com/kmousaa/PYDO
cd pydo
```

- [ ] Activate the venv
```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

- [ ] Install the packages
```bash
pip install -r requirements.txt
```

- [ ] Run PYDO
```bash
python pydo.py
```

You’re good to go.
